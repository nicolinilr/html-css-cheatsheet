Luke's Cheatsheet to HTML and CSS

Setting Up Your index.html

The Boilerplate Code

There are several bits of code that go in most any html site. The first is at the very top, where we let the browser know what type of file it is by inserting <!DOCTYPE html>. Remember this at the very beginning!

The next bit may seem obvious, but is easy to forget: the <html> tag! Make sure that this tag (opening and closing!) wraps around all your html content. It is also good to remember to add a language tag, like <html lang='en'> for English, to help with SEO and with user needing translations to know what language your page is in!

The Head

The head is the part of your html file that stores the relevant data for browsers and the page to function! This is where you put links, the title to display the name in the browser tab, etc!

First things first, lets get your CSS hooked up! To do that, just put a little piece of code into your head that looks like <link rel='stylesheet' type='text/css' href='./styles.css'/>, making sure your directory path to your stylesheet is correct. EZ-PZ!

Don't forget your <title>! Just put text between the opening and closing tags, and that will show up as your browser tab text!

Lots of other stuff can go here (hit F12 on any given website and look at the head, it is LOADED), but for now this is everything we need to get this cheatsheet website rolling!

The Body

The Body Tag contains all of the content you want to display onto the page! Using semantic HTML, the body itself can be broken up into several bits:

  • The Header (Different than The Head!)
  • The Nav Bar
  • Main and Articles
  • Asides
  • The Footer
  • And Probably More!

That's a lotta stuff and I want to get back to practicing React, so we're gonna go through em real quick-like starting NOW!

The Header

Top of your Page! Put you banner and whatever here! Wrap it with <header>!

The Nav Bar

Put links to navigate around your multi-page website! Or even on the same page if it's long! Wrap with <nav>, likes to contain lists!

Main and Articles

Use <main> to wrap around the main part of your webpage! Only use it once! And use <article> to wrap any relevant connected bits together! This is mostly used for context, especially helpful for accessibility!

Asides

Want to include a little extra info that's not necessary to understand the article? Use an <aside> tag!

Asides can be very useful for random fun facts, such as: did you know, raw pineapple has digestive enzymes that try to eat you while you're eating it!!

The Footer

Kinda like a header but at the bottom! Put career links, copyright info, whatever down here, stuff people might want to find but isn't so important it needs to be shown off. Wrap it with a <footer> tag.

And More!

There's <code>, <strong>, and lots of other stuff probably! Google it you nerd!

Let's Talk About Tables

Crazy Stuff! To make one, start by wrapping with a <table>, then each row is wrapped by <tr>. In each row you put the data in <td>, except the headers which are in <th>.

Setting Up Your styles.css

The Boilerplate Code

There really isn't any! Just start slapping down selectors for all your fun stuff. You can use the *{} selector to set base-level stuff you want everything to have, if that's sometimes useful! The html {} and body {} selectors can be useful in similar ways! Try setting font-size in one of these, and using em instead of px later, makes it way easy to rescale stuff for changing screen sizes!
